Классификатор дерева по его листу (Telegram Bot)

GitHub: https://github.com/saganvs/tree_classification_bot
Telegram bot: http://t.me/TreeClassBot

Команды бота

/start - начать работу с ботом
/help - вывести список доступных команд
/model - вывести список доступных моделей для классификации
/trees_list - вывести список известных боту деревьев

Как пользоваться ботом

  1. Отправьте боту сообщение /start
  2. Выберите модель для классификации (или оставьте модель по умолчанию)
  3. Отправьте боту фотографию с листом на светлом нейтральном фоне
  4. Если классификация удалась, бот отправит сообщение с результатом

Обученные модели

  1. K-Nearest Neighbors
    sklearn: KNeighborsClassifier
  2. Decision Tree
    sklearn: DecisionTreeClassifier
  3. Gaussian Naive Bayes
    sklearn: GaussianNB
  4. Random Forest
    sklearn: RandomForestClassifier
  5. C-Support Vector Classification
    sklearn: SVC

Как поставить бота себе

  1. Установите необходимые библиотеки:
    • python 3.8.6
    • python-telegram-bot 13.1
    • scikit-learn 0.22.2.post1
    • scikit-image 0.18.1
    • numpy 1.19.4
    • opencv-python 4.4.0.46
  2. Скачайте бота из репозитория
  3. В директории telegram/RU/ создайте файл t.txt и запишите в него токен вашего бота
  4. Через интерпретатор питона запустите файл RUBot.py
